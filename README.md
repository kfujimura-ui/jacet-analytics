## 自動化について

Claudeデスクトップアプリのスケジュールタスクで自動収集・生成。  
実行日にアプリが起動しており、ChromeでGA4にログイン済みであることが条件。

- GA4プロパティ：`p524091281`
- 前日18:00・当日8:00にGoogleカレンダー＋Claudeからリマインド通知あり

## データ仕様

- **計測開始：** 2026年2月（GA4設置以降）
- **中国からのアクセス：** エンゲージメント率12%・平均滞在3秒 → ボット判定・除外
- **コンバージョン（入会完了）：** GA4キーイベント未設定のため現在計測不可
